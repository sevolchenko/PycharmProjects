Game Rules

The original logic game, which is not common, can be conditionally called a two-dimensional Rubik's Cube. It is necessary to build vertical lines from cubes of the same color. A very difficult logic game.

Keyboard Bindings

Move the blue square with the keyboard arrows, and with the X and Z keys, rotate this square with the cubes that hit it.
